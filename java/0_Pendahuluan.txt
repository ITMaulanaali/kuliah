Compiler java sebenarnya berfungsi sebagai Interpreter. Yaitu tidak seperti c++ yang benar benar dirubah ke machine code untuk dijalankan mesin(komunikasi hardware lewat kernel)
Java menggunakan Interpreter yang menjalankan Program java dari hasil ByteCode. Kemudian interpreter akan berkomunikasi dengan API sistem untuk berkomunikasi dengan kernel.

Pada Bahasa pemrograman java. Ini secara implisit atau otomatis mengimport Package standar default atau bawaannya yaitu java.lang
