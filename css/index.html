Selector

    Simple Selector

  • Type

    • Contoh merubah judul dengan type h1 diatas dengan warna deepskyblue menggunakan type selector

  • Class

    • Contoh merubah text HELLO dan DEEPPINK ini dengan warna deeppink menggunakan class selector

  • Id

    • Contoh merubah kalimat ini dengan warna blueviolet menggunakan id selector

    Combinator Selector

  • Child

    • Contoh merubah child yang memiliki child bold yang italic HALLO BOLD DIDALAM ITALIC ini dengan warna merah menggunakan child selector

  • Descandent

    • Contoh merubah child kata italic HALLO dan GREEN ini dengan warna hijau menggunakan descandent selector

  • Adjacent Sibling

    • Contoh merubah paragraf setelah judul di paling atas yang menggunakan type h1 dengan warna brown. Paragraf kedua tidak akan terpengaruh karena hanya yang disampingnya atau dekat

  • General Sibling

    • Contoh merubah paragraf setelah h2 yaitu judul Selector entah seberapa jauh jaraknya dengan warna orange. Ini akan merubah semua turunanya (paragraf apapun setelah h2) menjadi warna orange

Color

Pemilihan warna menggunakan berbagai macam set color

  • Color Name

    • Contoh merubah border ini menjadi warna aqua dengan jenis pemilihan warna menggunakan color name

  • Hex Code

    • Contoh merubah warna border ini menjadi warna merah dengan jenis pemilihan warna menggunakan hex code

  • RGB Value

    • Contoh merubah warna border ini menjadi warna hijau dengan jenis pemilihan warna menggunakan rgb value

  • RGBA Value

    • Contoh merubah warna background ini menjadi warna hijau dengan jenis pemilihan warna menggunakan rgba value dan set opacitynya dengan setengah solid

  • HSL Value

    • Contoh merubah warna background ini menjadi warna hijau dengan jenis pemilihan warna menggunakan hsl value dan set kemurniannya/saturasi 100% dengan kecerahan/lightness 50%

Font

  • Font Family

    • Contoh merubah font kalimat ini menjadi verdana

  • Font Size

    • Contoh merubah ukuran/size kalimat ini dengan extra large (xx-large)

    Contoh merubah ukuran/size kalimat ini dengan extra small (xx-small)

    Contoh merubah ukuran/size kalimat ini sebesar 30px

    Contoh merubah ukuran/size kalimat ini sebesar 50%

    Contoh merubah ukuran/size kalimat ini dengan setengah ukuran menggunakan em

  • Font Weight

    • Contoh merubah ketebalan kalimat ini dengan ketebalan bold 1/solid menggunakan font-weight

  • Font Style

    • Contoh merubah gaya kalimat ini dengan gaya italic/miring menggunakan font-style

Text

  • Text Transform

    • Contoh merubah kalimat ini dengan huruf kapital/uppercase menggunakan text transform

  • Text Decoration

    • Contoh menambahkan kalimat ini dengan garis bawah/underline menggunakan text decoration

  • Text Align

    • Contoh menempatkan kalimat ini ke tengah/center menggunakan text align

  • Text Shadow

    • Contoh memberikan efek shadow pada kalimat ini menggunakan text shadow dengan nilai ke kanan 10px kebawah 5px kesolid an 5px dengan warna bayangan hitam/black

  • Word Spacing

    • Contoh meregangkan antar kata pada kalimat ini sebesar 0.5em menggunakan word spacing

  • Letter Spacing

    • Contoh meregangkan antar huruf pada kalimat ini sebesar 5px menggunakan letter spacing

Display

  • Inline

    • Contoh kata HALLO ini akan di display menggunakan inline sehingga akan selalu mengikut letak layout sesuai kata sebelumnya

  • Block

    • Contoh kata HALLO ini akan di display menggunakan block sehingga selalu ditampilkan sendiri tanpa ada kata yang mengikut
