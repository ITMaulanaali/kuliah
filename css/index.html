HTML untuk pembelajaran CSS

Semua text di page ini sudah di set font nya menjadi Times New Roman dengan universal selector css

Berikut adalah kategori contoh pembelajaran CSS

  • Selector

    • Selector

    • Type Selector

      • Contoh merubah judul dengan type h1 diatas dengan warna deepskyblue menggunakan type selector


    • Class Selector

      • Contoh merubah text

      HELLO

      dan

      DEEPPINK

      ini dengan warna deeppink menggunakan class selector


    • Id Selector

      • Contoh merubah kalimat ini dengan warna blueviolet menggunakan id selector


    • child Selector

      • Contoh merubah child yang memiliki child bold yang italic HALLO BOLD DIDALAM ITALIC ini dengan warna merah menggunakan child selector


    • descandent Selector

      • Contoh merubah child kata italic HALLO dan GREEN ini dengan warna hijau menggunakan descandent selector
